# Sigrún

Testing engine to experiment with C# and game development using "Veldrid" for the 
graphics API.

This engine is built to work with [SCP Containment Breach](https://www.scpcbgame.com/)
assets due to their accessibility. As such, this project is licensed under Creative Commons
